
# URL Rewrite Sample


## How to setup:
--------------

 1. Add HOSTS entries

    127.0.0.1	Area1.com
	127.0.0.1	Area2.com

 2. Add IIS Application
	Create 2 bindings to site/application: 
	http port 80 -> www.Area1.com 
	http port 80 -> www.Area2.com
3. Make sure that URL Rewrite 2.1 module is installed
